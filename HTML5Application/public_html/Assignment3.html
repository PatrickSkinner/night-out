To use the emulator, link the application's script name in the index.html file and load the index.html to run it

How the Application works

Load the Index.html file, this runs the emulator which in turns loads the application. The application is simple in use, First off it gives you four different options, this includes Food, Pub, Clubs and Taxi. You can click on any of them which loads up the next list. This includes a set of different options that the user can select, this shows a link to the selected option, and a directions button that loads a map with directions to the option from your current position. Back buttons are on every screen allowing the user to get back to the first four main options.

The biggest frustration I faced while designing was that the bright colors would sometimes make intricate details in the app difficult to see because of its small UI.
